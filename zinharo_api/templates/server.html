Want to help the cracking cause?

Zinharo is designed to be a lightweight distributed computing framework, allowing servers or computers of any kind to contribute to cracking pcap files. If you would like to help us with these efforts and flaunt your server power, you may download the Zinharo Client located below:



Setup is easy, to signup to the Zinahro network just add a username by setting the `ZINHARO_USERNAME` environment variable and the same for the password with `ZINHARO_PASSWORD`. The final environment variable that needs to be set is `ZINHARO_SIGNUP`, which has to be "true" (e.g. on Linux: `export ZINHARO_SIGNUP=true`). The rest is a completely automated process and can simply be left on using nohup/windows equivalent!
